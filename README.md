# Grehasoft PMS & CRM

A Project Management System and CRM built using:

Frontend:
React + TypeScript

Backend:
Python Django REST Framework

Database:
MySQL

Features:
- Dashboard overview
- Project management
- Task management
- Kanban board
- Client management
- Lead management (CRM)
- Lead to project conversion
- User & role management
- Department management
- Activity logs
- Profile management
