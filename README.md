# Grehasoft PMS & CRM

Grehasoft PMS is a Project Management System with CRM functionality.

## Tech Stack

Frontend
- React
- TypeScript
- Axios
- Bootstrap

Backend
- Python Django
- Django REST Framework

Database
- MySQL

## Features

Dashboard
- Project overview
- Task progress tracking

Project Management
- Create and manage projects
- Assign project managers
- Track project progress

Task Management
- Create tasks
- Assign users
- Task status tracking
- Kanban board

CRM Module
- Lead management
- Lead assignment
- Lead conversion to project
- Client creation

Admin Management
- User management
- Roles and permissions
- Department management
- Task types

Other Features
- Activity logs
- Profile management
- Password change
